## <a id="es">Descripción en Español</a>

### ¿Qué es Musicfy?
Musicfy es una aplicación Android que busca animar a la gente descubrir nueva música, está desarrollado en 
Android Studio con Kotlin y usa Firebase como base de datos en la nube.

La idea de Musicfy surgió a raíz de mi proyecto final de grado y a día de hoy sigo dedicandole tiempo
porque me divierte seguir añadiendo características y quiero ver a donde soy capaz de llevarlo, es un desafio para mi mismo.

### Características clave

#### Autenticación de usuarios:
La aplicación cuenta con un inicio de sesión y una opción para crear cuentas, además de dar la opción de recuperar la cuenta en caso de no recordarla.

Una vez iniciada la sesión se puede ver la pantalla principal, donde se listan todos los álbumes de la base de datos filtrados por género.

## <a id="en">English Description</a>

### What is Musicfy?
Musicfy is an Android application designed to encourage people to discover new music. It was developed in Android Studio using Kotlin and leverages Firebase as its cloud database.

The idea for Musicfy originated from my final degree project, and it has since become a passion project. I continue to develop it because I enjoy adding new features and challenging myself to see how far I can take it.

### Key Features
#### User Authentication:
Musicfy offers a seamless sign-in experience, account creation options, and an account recovery feature for forgotten credentials.

<img align='center' src="https://github.com/ManelTar/Musicfy/blob/Readme-img/Screenshot_20241022_211507_Musicfy.jpg" width="180"> <img align='center' src="https://github.com/ManelTar/Musicfy/blob/Readme-img/Screenshot_20241022_211515_Musicfy.jpg" width="180">

#### Album Discovery: 
Once logged in, users can explore a home screen listing all albums in the database, filtered by genre. Clicking on an album provides additional details, with the option to leave comments or mark it as a favorite.

<img align='center' src="https://github.com/ManelTar/Musicfy/blob/Readme-img/Screenshot_20241022_211530_Musicfy.jpg" width="180"> <img align='center' src="https://github.com/ManelTar/Musicfy/blob/Readme-img/Screenshot_20241022_211535_Musicfy.jpg" width="180">

#### Search Functionality:
Musicfy includes a search tab where users can browse all albums by default, or filter results by album name, artist, or genre using the search bar.

<img align='center' src="https://github.com/ManelTar/Musicfy/blob/Readme-img/Screenshot_20241022_211539_Musicfy.jpg" width="180">

#### Social Features: 
Musicfy includes social network-like features such as a favorites section, comment sections, and a feed where users can view others' comments on various albums.

<img align='center' src="https://github.com/ManelTar/Musicfy/blob/Readme-img/Screenshot_20241022_211541_Musicfy.jpg" width="180"> <img align='center' src="https://github.com/ManelTar/Musicfy/blob/Readme-img/Screenshot_20241022_211553_Musicfy.jpg" width="180"> <img align='center' src="https://github.com/ManelTar/Musicfy/blob/Readme-img/Screenshot_20241022_211545_Musicfy.jpg" width="180">

#### Personalization: 
Users can customize their experience by selecting a profile picture and changing the app's theme to match their preferences.

### Future Enhancements
Looking ahead, I plan to add new features such as a friend system where users can add and search for others, view their favorite albums, and read their comments.

Additionally, I aim to expand Musicfy's reach by developing an iOS version in Swift, allowing a broader audience to enjoy the app.

This project showcases my ability to build and evolve a complex, full-stack mobile application from the ground up, with a focus on both backend and frontend development, user experience, and cloud services integration. The continuous evolution of Musicfy demonstrates my commitment to learning, problem-solving, and delivering a polished product.
